This website is meant to act as a light-weight calorie tracker to keep you on course through the day/week/month/year. I say it this way (day/week/month/year) because there are no user accounts. There is no database storing your data. All data is stored right in your browser and it never goes further than that.

The code is publicly available on GitHub

This website was written in TypeScript using Angular2 and HTML5 with a mobile-first design

Current features

  • Add Meal Days, representing a single day in which meals were consumed
  • Add Meals, meals are added to days and comprised of different foods
  • Add Foods, foods are added to meals
  • Page displayed Meal Days calculates total consumed macros (calories, protein, carbs, fat) alongside the limits set for yourself that day

Upcoming Features

  • Save food for re-use in multiple meals
  • Delete/Update Food from a Meal
  • Delete/Update Meals
  • Delete/Update Meal Days
